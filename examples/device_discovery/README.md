# Device Discovery

Demonstrates:
- Full device discovery using `DeviceDiscoveryEngine`
- Discovering all devices with timing info
- Filtering by vendor (Qualcomm)
- Probing a specific USB device descriptor
- Displaying detailed device information
- Iterating over device capabilities

## Build

```bash
cd build
cmake --build . --target example_device_discovery
```

## Run

```bash
./example_device_discovery
```
