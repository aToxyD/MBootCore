# Basic Connect

Demonstrates:
- Creating a Runtime via `RuntimeFactory::createDefault()`
- Setting up callbacks and logging with `ConsoleLogger`
- Initializing the runtime
- Discovering devices with a custom timeout
- Handling discovery results and errors
- Retrieving SDK version info
- Clean shutdown

## Build

```bash
cd build
cmake --build . --target example_basic_connect
```

## Run

```bash
./example_basic_connect
```
